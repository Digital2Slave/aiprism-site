# Edge AI · 端侧智能

本专栏聚焦 **端侧 AI（Edge AI）** 的工程实践，围绕模型在 **手机、嵌入式设备、工业终端** 上的高效运行展开。

## 关注方向

- 端侧模型推理与优化（INT8 / FP16 / 混合精度）
- Android / Linux / 嵌入式 AI 部署
- RKNN / NCNN / TensorRT / OpenVINO
- 摄像头 → 推理 → 业务闭环
- 功耗、时延、稳定性权衡

## 适合读者

- Edge AI / 端侧 AI 工程师
- 移动端 / 工业视觉开发者
- 关注 **模型落地** 的技术人员

> Edge AI 不是“缩水版 AI”，而是工程约束下的最优解。
>
